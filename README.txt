pour trouver la ligne qui part le plus tôt possible après Horaire parmi les lignes qui vont de l’Arret1 à l’Arret2 entrez :
ligtot(Arret1, Arret2, Ligne, Horaire)
pour trouver la ligne arrive le plus tardpossible avant Horaire parmi les lignes qui vont de l’Arret1 à l’Arret2 entrez :
ligtard(Arret1, Arret2, Ligne, Horaire)
